**CitationFormatter class** (src/rag/citation_formatter.py):

**Core method**:
```python
def format_citations(sources: List[Dict[str, Any]]) -> str:
    """
    Format source documents into citation text
    
    Args:
        sources: List of search results from SearchEngine
    
    Returns:
        Formatted citation string in Chinese, e.g.:
        
        ---
        来源：
        [1] TikTok广告投放策略.md (相关度: 0.85)
           "TikTok 广告投放的最佳时间是晚上 8-10 点，这个时段用户活跃度最高..."
        
        [2] AI工具应用指南.md (相关度: 0.72)
           "使用 ChatGPT 可以快速生成产品描述，提升转化率..."
    """
```

**Implementation details**:
1. **Handle empty sources**: Return empty string if sources list is empty
2. **Deduplicate**: Group sources by file_name, keep highest score per file
3. **Sort**: Order by relevance score (highest first)
4. **Format each citation**:
   - Number: [1], [2], [3]...
   - File name: From metadata['file_name']
   - Score: Format as 0.XX (2 decimal places)
   - Snippet: First 200 chars of content + "..." if truncated
5. **Build output**: "---\n来源：\n" + formatted citations

**Helper function**:
```python
def format_citations(sources: List[Dict]) -> str:
    """Convenience function wrapping CitationFormatter"""
    formatter = CitationFormatter()
    return formatter.format_citations(sources)
```

**Integration with QueryEngine** (modify src/rag/query_engine.py):

Update QueryEngine.query() to append citations to answer:
```python
def query(self, user_query: str) -> Dict[str, Any]:
    # ... existing retrieval and generation logic ...
    
    # Format citations
    from .citation_formatter import format_citations
    citations = format_citations(sources)
    
    # Append to answer if sources exist
    if has_sources and citations:
        answer = answer + "\n\n" + citations
    
    return {
        'answer': answer,
        'sources': sources,
        'has_sources': has_sources
    }
```

**Citation format rationale**:
- Chinese language matches user's wiki and Henry's responses
- Numbered citations [1], [2] enable easy reference in conversation
- Relevance scores help user assess source quality
- Content snippets provide context without overwhelming
- 200 char limit balances detail vs readability
- Deduplication prevents redundant citations from same file

**Why append to answer not separate field**: Frontend (Phase 7) can display citations inline with answer text. Simpler integration than managing separate citation UI components.
