1. Add request/response models to src/api/models.py:
   - DocumentResponse(BaseModel): filename: str, file_path: str, topic: str, modified_at: str, size_kb: float
   - DocumentListResponse(BaseModel): documents: List[DocumentResponse], total: int
   - TopicDocumentsResponse(BaseModel): topics: Dict[str, List[DocumentResponse]]
   - SearchRequest(BaseModel): query: str, top_k: int = 5, min_score: float = 0.5
   - SearchResultItem(BaseModel): content: str, score: float, document_path: str, chunk_id: str
   - SearchResponse(BaseModel): results: List[SearchResultItem], query: str, total: int

2. Create src/api/routes/documents.py with APIRouter and endpoints:

   **Router setup:**
   - Create APIRouter with prefix="/api/documents", tags=["documents"]
   
   **Endpoints:**
   
   a) GET /api/documents - List all documents
      - Query params: None (future: pagination, filtering)
      - Response: DocumentListResponse (200 OK)
      - Dependencies: 
        * search_engine: SearchEngine = Depends(get_search_engine)
        * document_store: DocumentStore = Depends(get_document_store)
      - Logic:
        * Call search_engine.list_all_documents() to get document list
        * Call document_store.get_document_count() to get total count
        * Transform to DocumentListResponse format
      - Return documents ordered by modified_at DESC (newest first)
   
   b) GET /api/documents/topics - List documents grouped by topic
      - Query params: None
      - Response: TopicDocumentsResponse (200 OK)
      - Dependency: search_engine: SearchEngine = Depends(get_search_engine)
      - Logic:
        * Call search_engine.list_documents_by_topic()
        * Transform to TopicDocumentsResponse format
        * Each topic contains list of DocumentResponse objects
      - Return topics in alphabetical order
   
   c) GET /api/documents/search - Search documents by content
      - Query params: 
        * query: str (required)
        * top_k: int = 5 (optional)
        * min_score: float = 0.5 (optional)
      - Response: SearchResponse (200 OK)
      - Dependency: search_engine: SearchEngine = Depends(get_search_engine)
      - Logic:
        * Validate query is not empty (raise HTTPException 400 if empty)
        * Call search_engine.hybrid_search(query, top_k, min_score)
        * Transform results to SearchResultItem format:
          ```python
          SearchResultItem(
              content=result["content"],
              score=result["score"],
              document_path=result["metadata"]["file_path"],
              chunk_id=result["node_id"]
          )
          ```
        * Return SearchResponse with results and query metadata
      - Error handling: Empty query returns 400 Bad Request
   
   d) GET /api/documents/stats - Get document statistics
      - Query params: None
      - Response: Dict with document counts and topics
      - Dependencies:
        * search_engine: SearchEngine = Depends(get_search_engine)
        * document_store: DocumentStore = Depends(get_document_store)
      - Logic:
        * Call document_store.get_document_count() for file type counts
        * Call search_engine.list_documents_by_topic() for topic counts
        * Return: {"total": int, "by_type": {...}, "by_topic": {...}}

3. Update src/api/main.py:
   - Import documents router: `from src.api.routes import documents`
   - Include router: `app.include_router(documents.router)`
