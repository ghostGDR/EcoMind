1. Create src/api/routes/__init__.py (empty file for package)

2. Create src/api/routes/conversations.py with APIRouter and endpoints:

   **Router setup:**
   - Create APIRouter with prefix="/api/conversations", tags=["conversations"]
   
   **Request/Response models (add to src/api/models.py if not already there):**
   - CreateConversationRequest(BaseModel): title: str = "新对话"
   - CreateConversationResponse(BaseModel): id: int, title: str, created_at: str
   - ConversationListItem(BaseModel): id: int, title: str, created_at: str, message_count: int
   - ConversationListResponse(BaseModel): conversations: List[ConversationListItem], total: int
   
   **Endpoints:**
   
   a) POST /api/conversations - Create new conversation
      - Request body: CreateConversationRequest
      - Response: CreateConversationResponse (201 Created)
      - Logic: Call manager.start_conversation(title), return conversation metadata
      - Error handling: Catch exceptions and return 500 with error detail
   
   b) GET /api/conversations - List all conversations
      - Query params: None (future: pagination with skip/limit)
      - Response: ConversationListResponse (200 OK)
      - Logic: Call manager.list_conversations(), transform to response format
      - Return conversations ordered by created_at DESC (newest first)
   
   c) GET /api/conversations/{conversation_id} - Get conversation details
      - Path param: conversation_id: int
      - Response: ConversationResponse (200 OK) or 404 Not Found
      - Logic: Call manager.get_conversation(conversation_id)
      - If conversation is None, raise HTTPException(status_code=404, detail="Conversation not found")
      - Transform Conversation dataclass to ConversationResponse Pydantic model
      - Include all messages with sources in response
   
   **Dependency injection:**
   - All endpoints use: `manager: ConversationManager = Depends(get_conversation_manager)`
   
   **Response transformation:**
   - Convert Conversation dataclass to ConversationResponse:
     ```python
     ConversationResponse(
         id=conv.id,
         title=conv.title,
         created_at=conv.created_at,
         messages=[
             MessageResponse(
                 id=msg.id,
                 role=msg.role,
                 content=msg.content,
                 created_at=msg.created_at,
                 sources=msg.sources
             )
             for msg in conv.messages
         ]
     )
     ```

3. Update src/api/main.py:
   - Import conversations router: `from src.api.routes import conversations`
   - Include router: `app.include_router(conversations.router)`

4. Create tests/test_conversation_routes.py with integration tests:
   - Use TestClient from fastapi.testclient
