**实现策略（简化版 - v1）：**

1. **语义搜索** - 调用 self.semantic_search(query, top_k=top_k*2, min_score=min_score)
   - 获取语义相关结果

2. **关键词增强** - 检测查询中的技术术语
   - 定义技术术语列表：['ROI', 'CPC', 'CPM', 'CTR', 'conversion', '转化率', 'TikTok', 'AI', 'ChatGPT']
   - 如果查询包含这些术语，提升包含该术语的结果的权重

3. **结果合并**
   - 遍历语义搜索结果
   - 如果结果的 content 或 metadata 包含查询中的技术术语，score *= 1.2（提升 20%）
   - 按 score 降序排序
   - 返回前 top_k 个结果

**关键词检测函数：**
```python
def _extract_keywords(self, query: str) -> List[str]:
    """提取查询中的技术术语"""
    keywords = ['ROI', 'CPC', 'CPM', 'CTR', 'conversion', '转化率', 
                'TikTok', 'AI', 'ChatGPT', 'Facebook', 'Google']
    
    found = []
    query_lower = query.lower()
    for keyword in keywords:
        if keyword.lower() in query_lower:
            found.append(keyword)
    
    return found

def _boost_keyword_matches(self, results: List[Dict], keywords: List[str]) -> List[Dict]:
    """提升包含关键词的结果的评分"""
    for result in results:
        content_lower = result['content'].lower()
        metadata_str = str(result['metadata']).lower()
        
        for keyword in keywords:
            if keyword.lower() in content_lower or keyword.lower() in metadata_str:
                result['score'] = min(result['score'] * 1.2, 1.0)  # 提升 20%，最高 1.0
                break  # 每个结果只提升一次
    
    return results
```
